About the project

The project covers valuation topics over various financial instruments. So far, it mainly focuses on valuing equity and derivatives using both fundamental and technical analyses. The reason I learned HTML/CSS from scratch to build this website is to attach valuation machines (in Python) with my explanation. And I hope it will make valuation techniques and complicated financial products more comprehensive to audience, especially those don't have finance background.
